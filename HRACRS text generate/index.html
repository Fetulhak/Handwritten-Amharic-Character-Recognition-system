Handwritten Amharic Character and Word Recognition System

Drag and drop your handwritten document to our system using the drag and drop web interface provided ,then click recognize to get ouput of your handwritten documment in text form. You can copy the output to your word file for further editorial work.

A machine learning model trained against the manually collected handwritten documents and preprocessed dataset will classify the handwritten document. The editable text format will be displayed as a result.
